从《如何阅读一本书》开始（1）

发表于 2017-11-07 | 阅读次数:

阅读技巧

检视阅读

目的：

  1. 短时间了解一本书（即系统的略读）；
    步骤：

    1. 先看标题&副标题（如果有序就先看序），进行归类（比如“心理学”）；
    2. 再看宣传文案，了解书的主题、写作对象、核心问题；
    3. 三看目录，了解书的架构，从该架构中找出与主题特别有关的章节（关键章）开始浏览，并注意本章摘要&本章总结；
    4. 最后左翻翻右翻翻（有时一两段，有时一两页），凭兴趣、问题意识了解大概；如果没有兴趣，在第三步时就可以停下来了。
      • 前言、索引
      • 重点、摘要
      • 开头、结尾
      • 主要内容…

    如果觉得这本书较难，并且你有兴趣掌握（或者对你很重要），就进行下面的“粗浅的阅读”。

    ps：上面步骤的核心是在翻阅的过程中形成一种作者的思路，耗时约15分钟左右。

  2. 为分析阅读做铺垫（即粗浅的阅读）；

    步骤：

    1. 从头到尾过一遍，不停顿，不纠结，思绪中断处用笔画出，将眼光着重于那些你读得懂的地方；

    2. 还想了解，再读一遍，此时你会发现，原先没读懂的地方，比之前稍稍能多懂些了（这部分运用得更多的应该是分析阅读中的列纲要）；

    3. 速度技巧：为防止你的眼光停留在书籍的某一处过长时间，通过手指的带动（如果是软件的话使用软件的自动翻页功能），来提升速度；手指的速度须稍稍快过眼睛的速度。

      但速读不能让你很好的“理解”一本书，如果你想更好的“理解”一本书，就需要进入“分析阅读”阶段了。

分析阅读

  1. 第一阶段，了解书本说了什么（“整理内容大纲”）：

    步骤：

    1. 分类，越早越好；

      分类标准如下：

      • 虚构文学类；
      • 论证说明类；
        • 实用类（如何做）；ps：一些社科类的教科书也是实用类的书籍。
        • 理论类（是什么）。

    2. 用最简短的句子说出整本书在谈些什么；

    3. 按顺序与关系列出全书重要部分，此为全书纲要，再将各部分纲要也列出来。ps：这是为了找出作者是怎么说的。

    4. 找出作者在问，或想要解决的问题。如果说前两条是找出答案，那么这一条就是要我们在无法确定答案时，通过寻找问题来凸显答案。

  2. 第二阶段，与作者交流（ps：可以理解成是上面第四点的深化。）（“诠释一本书的内容”）：

    步骤：

    1. 诠释作者使用的关键字，与作者达成共识；ps：理解作者使用的术语、单字的确切意义（即字义），并在该前提下一起探讨接下来的问题与答案。
    2. 找出关键句，归纳相关主旨（即观点、大意、答案、知识点，字构成句，关键字可能构成关键句，联系后面，关键句的组合，可能就构成了作者的论述）；
    3. 找出作者的论述（即“主旨群”（是段落也不是段落，不是段落式因为它可能分散在各个地方），多个主旨可能经过语言的整合，指向同一个目的（大观点）。论述是形式，主旨是实质，多条论述可能指向一个主旨，而一条论述也可能指向多个主旨），重新架构这些论述的前因后果，以明白作者的主张；找论述的一些技巧：
      • 记住，所有的论述都包含了一些声明；这些声明有的是结论，有的是这些结论的理由；
      • 要区别出归纳、演绎这两种论述的不同之处；
      • 找出作者认为哪些事情是假设，哪些是有根据的，那些是不证自明的，这些都是作者论述的开端。
    4. 确定作者已解决哪些问题，未解决哪些问题。未解决的问题中，哪些是作者认为自己无法解决的。
    5. 总的流程：共识（关键概念）——关键句——主旨——论述

因为没有草原，就忘了你是马

发表于 2017-11-04 | 阅读次数:

虽说喜欢音乐，却说不上是个”认真的乐迷“（马世芳语），我很少专心一意地听一首歌，更多时候是把它们作为做事时的BGM来对待，因此可以说，音乐对我来说，歌曲比歌词更重要。然而也有歌词突然冲上台前的时候。

这两天都在和拖延症斗争，很遗憾，节节败退，我又开始整理起iTunes音乐库了。随机播放一首开始吧，然后就是把网上搬运下来的无损资源一批批地从WAV转成苹果的无损格式，再又一批批地拖进iTunes里，之后就是调整音乐的标签。这个过程其实蛮无聊的，但是在拖延这方面，却又有极强的“趣味吸引力”。这我也不清楚究竟有什么“趣味吸引力”，但是嗣后看到主界面专辑封面整整齐齐的样子时，却也又说不清为什么要觉得没有“趣味性”了。

就在这种思绪若有若无之际，歌词就冲上台前了。是朴树的《the fear in my heart》，出自他的最新专辑《猎户星座》，是该专辑的第八首歌。你听他是这么唱的：

你也曾经追问
然后沉默
渐渐习惯谎言
并以此为荣
因为没有草原
就忘了你是马
你卑微的人生
从不曾犯错的
无聊的人生
能不能 彻底地放开你的手
敢不敢 这么义无反顾坠落
坠入黑暗中
坠入泥土中
的海阔天空

这种歌词特别有抓力。我一直觉得我常常是那种很小心翼翼的人，很在乎一个不存在的人的看法。“十目所视，十手所指”，就经常觉得自己的根本想法是错的。

骨子里有种犯罪欲。当然这究竟是否真的是犯罪，也不是，就是觉得自己的选择是不对的，进而，就是否定自己的选择了。然后发展到今天，就是否定这种否定本身了。笑。

还是前几天，我假设自己的生命期限是60年，然后算了算，发现也就剩下一万多天了。再不将初心重拾，放手一博，时间也要用完了。

说得真好，从明天开始吧。

必须得些什么？

发表于 2017-11-03 | 阅读次数:

上一次写博客，已经是五六年前的事了，期间也一直有继续写的想法，然而却终于还是没有施行。对我来说，开头太难了。

或许这就是拖延的缘故，因为对一些东西似乎因为过于看重，而不敢轻易尝试。然而，如果不写，我怕我真的要成为一个废人了。

脑海里有时没时的都会想到苏力先生的质问：你的贡献是什么？ 这么一想，我确实无为人的价值。这令我会在那仅有的几秒时光中，自责一下。

如果要重拾为人的价值，对于我来说，首先要做到的，就得是抓住时间，进而掌握命运。这飘忽的时间如何掌握？就得靠写作了。

是故，我的博客的写作，其首先要满足的不是读者，而是自身。这将是一连串与自我的对话，是对自我的探索。谁是我？我是谁？我相信，在此同时，价值也将因之而被创造。

我将不是孤立的。我将是被需要的，一个有价值的个体。

但这样的想法又将将自己拖入无边的拖延中，所以我还是选择这么想：

写或者不写，在我身上都不会有什么奇迹出现，这前后之间唯一的不同只在是：你是否由衷的喜欢你的选择？
